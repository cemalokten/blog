```bash
GET /calculate/add/1/2/3 -> 6

GET /calculate/subtract/10/5 -> 5

GET /calculate/divide/50/5 -> 10

GET /calculate/add/20/30/subtract/20/add/10/divide/2/multiply/2 -> 40
```
